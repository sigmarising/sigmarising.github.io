经过不断与 DeepSeek 辅助交流后，我归纳总结了一份适合我自己使用的“户外日志撰写格式”，在此简单做一下归纳记录（大部分文字为 AI 辅助生成，全部排版均为人工手动二次整理）。

我的记录载体是 *Rite In The Rain* 小号本（7.6 ✖ 12.7 厘米，线圈竖向翻页）。

## 1. 正面

现场记录，**只写观察所得，保持简单，快速记录**，*不做任何加工处理*。

### 1.1 Top：页眉

- 路线 地点 身份（领队/队员/独行）
- 日期 星期 天气

```
大寺~草甸环穿线 河北唐山 领队
2022/10/11 TUE 晴转多云
```

### 1.2 正文

各个区块的顺序**不必固定，也不必丝毫不差**，只是提供记录的**思路和视角**。

#### 区块 A：路线路况的实际感受

**这不是轨迹记录**，也不是路标流水账，**而是脚底的真实反馈**。\
回忆脚底的感觉：当时硌脚了吗？打滑了吗？哪段路走得最小心？那就是要写的。

```
滑 / 石板像泼了油
松针厚到踩下去弹不回来
风从左边推人，身体一直往右歪
碎石坡，每一步都在猜哪块石头是稳的

不写什么：
❌ 海拔1800米（轨迹有）
❌ 走了3.5公里（轨迹有）
❌ 路很陡（太抽象）
```

#### 区块 B：人/动物/景物的事情瞬间

**一个动作，胜过十句评价**。\
只写“他做了什么/说了什么”，不写“他很勇敢/他很累”。\
**动作是矿石，评价是加工品**——加工下山再做，在山里只采矿。

```
一队员过河腿软，蹲地笑说自己是爬过来的
收队捡了根鹰羽插包侧，走起来像旗
另一队员恐高，过石海全程摸石头，像在摸盲文

遇采药人，他说这山他爬了四十年
新鲜刨土，野猪刚来过
溪边有堆摆整齐的石头，不知谁垒的
```

#### 区块 C：意外发生时的决策/思考

**轨迹**只显示你走了 B 点没走 A 点，**不显示为什么**。\
任何“没按原计划走”的时刻，都是决策节点。哪怕只是犹豫了一下，也值得记。\
**一句话公式**：遇到了什么 + 想了什么 + 做了什么。

```
原定营地有牛群，往前推1km
垭口雾浓到能尝出水，等20min没散→全员表决下撤
看到新鲜脚印，跟了500m又折返——感觉方向不对
```

#### 区块 D：声音/气味/身体状态

**五感切片：照片拍不出的，全在这里**。\
**声音和气味**是记忆最强的锚点。\
三年后看到“牛铃从谷底浮上来”，那一瞬间的声音会自己响起来。

```
水声大到喊话靠吼
牛铃从谷底浮上来
柴烟 / 腐叶 / 前一晚没干的袜子
雾浓到能尝出水味

左膝微酸，第一次闪过“要不要求援”
渴到吞口水都剌嗓子
躺在花岗岩上，暖意从后腰渗进来
```

#### 区块 E：速写简笔画

线圈本翻着画，**画歪更好**。\
不用会画画：**精确是相机的事，手感是你的特权**。\
线条歪歪扭扭，正好是“当时手冻僵了”、“当时走得喘”\
——多年后看到这幅歪画，你连那天的温度都想得起来。

```
画什么：
- 独木桥（两根歪木头+波浪线）
- 这个垭口的山脊线
- 帐篷扎营的排列
- 那棵长得像鹿的树
- 采药人背篓的轮廓
```

## 2. 反面

出山/归来后的反思，是对正面所有记录文字的**加工提炼和回忆思考**。

### 2.1 正文

三个要点，帮助自己凝聚这段时空的模因。

#### 区块 X：当时不知道的事

*对着正面关键词，写一句“后来才明白”*。\
这是整页日志里最有嚼头的部分。\
正面是“发生了什么”，**这里是“后来才懂了什么”**。

```
过独木桥时全队屏息，没人说话。
后来那位女队员说，那十几秒她听见了自己的心跳。
——原来恐惧不是噪声，是专注。

过独木桥时腿软，心想完了没人捞我。
现在坐在办公室，最想重走的正是那根木头。
——恐惧原来比平坦更让人惦记。
```

#### 区块 Y：哪个瞬间后来在脑海里反复播放

那么多小时的山路，偏偏这个镜头卡住了。\
**如果只有一个瞬间能带下山**，是哪个？\
把它写下来，**这是你这一天浓缩成的晶体**。

```
云开雾散那五分钟，全队没人说话，都在按快门。
后来没人看照片，但那五分钟谁都记得。

老爷子说“我爬不动几年了”，说这话时没看山顶，看着我们来时的路。
```

#### 区块 Z：这次旅途，哪些人和事让自己记住了

**人物速写集**：不必多，2-3个，更少也无所谓。\
**人名可以忘，特征要留下**。

```
上海来的女孩，过河吓得尖叫，过完傻笑了半小时
新驴，第一次扎营，半夜钻出帐篷看银河，以为没人发现
向导——哼歌，调子被风撕碎了

采药人——烟锅在嘴角一明一灭，他说绝望坡是他歇脚的地方
那只獾——下山头灯照到它，它回头看我，像在说这是你家吗
```

### 2.2 领队/个人复盘专区

#### 区块 V：领队有谁

写全名或花名。\
一是责任记录，二是多年后你会想知道“当年和我一起带队的都是谁”。

```
主领：你
中队：小王
收队：老张
```

#### 区块 W：复盘总结

领队写传承，个人写状态。\
**这是你给下次的自己留的工具箱**。\
不是写给今天看的，是写给下次出发前的自己看的。

```
路线：碎石坡雨季易塌，建议绕山脊
人员：恐高的队员别放队尾，视线无遮挡反而更怕
装备：电工胶带能修鞋底，下次多带一卷
自己：今天压队压太死，前锋等太久

左膝：走平路不疼，下台阶微酸——今年少背重装
水：全程2.3L，比上次少0.5L刚好
害怕阈值：今天闪过三次“要不要求援”，都是在下坡
```

### 2.3 End：页脚

展望和写给自己的话。\
一行或三行，**下山那一刻最真实的念头**。\
**不用深刻，真就好**。多年后回看，最戳你的往往是这句。

```
下山吃了碗汇通面，醋放多了，但还是香的。
16人上山，16人下山。今日平安。
谢谢今天没下雨。
还会再来吗？应该会。
这山还在，我下次不一定在——所以这次要写下来。
```
